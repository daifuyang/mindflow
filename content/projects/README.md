# 项目计划

> vibe coding 项目管理目录。结合第一性原理，按照商业价值优先级排序。

## 核心理念

- **商业项目优先**：精力充沛时专注商业项目，最大化商业价值
- **累了转个人项目**：当疲惫或需要换脑子时，切换到个人项目作为调剂
- **第一性原理**：先问"这个项目解决了什么根本问题"，再评估价值

## 目录结构

```
projects/
├── commercial/     # 商业项目（优先）
│   ├── README.md   # 商业项目索引
│   └── *.md        # 各项目详细计划
│
└── personal/       # 个人项目（娱乐）
    ├── README.md   # 个人项目索引
    └── *.md        # 各项目详细计划
```

## 优先级原则

### 商业项目排序依据

1. **直接变现能力**：项目能否直接带来收入
2. **资产积累**：是否建立可复用的技术/品牌资产
3. **杠杆效应**：能否放大其他工作的效果
4. **进入门槛**：竞争壁垒高低

### 个人项目排序依据

1. **技术探索价值**：能否打开新的技术可能性
2. **学习投入产出比**：学到的技能能否迁移到商业项目
3. **有趣程度**：纯粹因为想做而做
