# zerocmf

> 通用 AI 编程架构，让开发者高效构建管理后台。

## 基本信息

- **优先级**：P0
- **状态**：进行中
- **第一性原理**：降低管理系统的开发成本，让 AI 辅助编程成为主流
- **GitHub**：https://github.com/daifuyang/yishan

## 核心技术

| 层级 | 技术栈 |
|------|--------|
| 前端 | React 19 + Ant Design Pro + Umi 4 |
| 后端 | Fastify 5 + Prisma 7 + TypeBox |
| 文档 | Docusaurus 3 |
| 组件库 | TipTap 3 + Radix UI |
| 部署 | 阿里云 FC3 |

## 当前进度

### 已完成

- [x] monorepo 架构（pnpm workspace）
- [x] 管理后台基础框架
- [x] 后端基础框架（JWT + 权限体系）
- [x] 核心模块：用户、角色、菜单、部门
- [x] 内容管理：文章、分类、页面、模板
- [x] 文件管理：本地存储、七牛云存储
- [x] 插件运行时架构

### 进行中

- [ ] 系统监控与日志采集

### 计划中

- [ ] AI 编程工作流集成

## 插件生态

zerocmf 采用插件化设计，以下插件正在规划：

- **CRM 插件**：客户管理、销售漏斗
- **商城插件**：商品、订单、支付
- **小工单插件**：工单流转、客服系统
