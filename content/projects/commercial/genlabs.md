# genlabs

> 内容创作者的 AI 工具平台，素材管理 + 内容创作 + 一键发布 + 在线配音。

## 基本信息

- **优先级**：P0
- **状态**：进行中
- **第一性原理**：让内容创作者一键发布内容到多平台，无需手动操作
- **地址**：https://mp.zerocmf.com

## 核心功能

| 功能 | 说明 |
|------|------|
| 素材库 | 图片、视频、音频素材管理 |
| 内容管理 | 文章、图文、视频内容编辑 |
| 一键发布 | 一次编辑，多平台同步发布 |
| 在线配音 | AI 文字转语音，支持多音色 |

## 技术架构

- 基于 zerocmf（yishan）构建
- 独立部署，共享基础架构
- 数据库：MySQL
- 缓存：Redis
- AI 能力：接入第三方 TTS API
