# 增长规划：涨粉 + 付费产品

> 本规划基于当前实际情况制定：兼职投入、粉丝从零开始、全栈技术背景、有开源项目积累。

## 核心策略

**一句话总结：用「AI + 开源实战」内容涨粉，用「yishan CMS SaaS」变现。**

两个目标不是分开做的，而是一条链：

```
技术内容 → 涨粉 → 信任感 → 开源用户 → 付费转化
```

---

## 第一阶段：内容起步（第 1-3 个月）

### 目标

- 各平台合计粉丝达到 **1000+**
- 建立稳定的内容发布节奏
- 确定哪类内容数据最好，找到爆款方向

### 平台选择

**主攻 2 个平台，精力有限不要铺太多：**

| 平台 | 优先级 | 原因 |
|------|--------|------|
| **掘金** | ⭐ 主攻 | 开发者精准用户，技术文章天然适合，SEO 好，长尾流量持续 |
| **微信公众号** | ⭐ 主攻 | 私域沉淀，粉丝粘性高，适合后续转化付费 |
| B站 | 辅助 | 录屏教程有长尾流量，但制作成本高，先不强求 |
| 小红书 | 暂不做 | 技术内容不是主流受众 |

### 内容策略

**每周产出 2 篇内容**（兼职可执行的节奏）：

- **周中 1 篇**：技术干货（掘金 + 公众号同步发）
- **周末 1 篇**：项目实战/开源日志（掘金 + 公众号同步发）

### 四大内容方向（按优先级排序）

#### 1. AI 辅助开发实录（最容易出爆款）

> 2025-2026 最热的话题，流量红利期

- 「用 AI 30 分钟搭建一个 XXX 系统」
- 「AI 写代码靠谱吗？我用它开发了一个月的真实体验」
- 「我让 AI 帮我重构了 5 年前的代码，结果...」
- 「Cursor/Trae 实战：从零到上线全过程」

#### 2. 开源 CMS 实战（建立专业形象）

> 直接给 yishan/ZeroCMF 带用户

- 「为什么我用 3 年做了 3 版 CMS」
- 「Next.js 全栈 CMS 架构设计」
- 「Prisma + PostgreSQL 实战踩坑指南」
- 「开源 CMS 怎么设计才好用？」

#### 3. 全栈技术教程（长尾 SEO 流量）

> 搜索流量持续带粉

- 「Next.js App Router 最佳实践 2026」
- 「shadcn/ui 从入门到实战」
- 「Fastify vs Express：2026 年该选哪个？」
- 「Prisma 完全指南：从建模到部署」

#### 4. 独立开发者日志（建立人格魅力）

> 让人关注你这个人，而不只是技术

- 「全栈独立开发者月报」
- 「我的开源项目第一个 100 star」
- 「兼职做独立开发的时间管理」

### 内容制作流程（每篇控制在 2-3 小时内）

1. **选题**（15min）：从日常开发中发现选题，记录到 MindFlow
2. **写作**（1.5h）：用 AI 辅助起草，自己修改润色
3. **配图**（15min）：代码截图 + 架构图（用 Excalidraw）
4. **发布**（15min）：掘金 + 公众号同步发布
5. **复盘**（下次写之前看上篇数据）

### 付费产品路线（按推出顺序）

#### 产品 1：yishan Cloud（SaaS 托管版）

> 最高优先级，直接复用你的核心技术资产

- **定位**：开箱即用的 CMS 托管服务，注册即用，无需部署
- **定价**：免费版（1 个站点）+ Pro 版（¥99/月，多站点 + 自定义域名 + 优先支持）
- **技术**：多租户架构，部署在阿里云 FC
- **推广**：通过开源用户自然转化

#### 产品 2：付费模板/主题包

> 低成本高利润的数字商品

- **定位**：基于 yishan 的精美企业模板
- **定价**：¥99-299/套
- **内容**：企业官网模板、个人博客模板、产品展示模板
- **推广**：在 yishan 文档和 GitHub 中推荐

#### 产品 3：技术咨询/定制服务

> 高客单价，但不可规模化，作为补充收入

- **定位**：基于 ZeroCMF 的企业定制开发
- **定价**：¥500/小时 或按项目报价
- **渠道**：公众号菜单 + 个人网站

## 关键原则

1. **先做内容，再做产品** — 没有粉丝基础就做产品是浪费
2. **内容即推广** — 每篇文章都要自然带出 yishan/ZeroCMF
3. **AI 是杠杆** — 用 AI 提高写作效率，2 小时完成一篇高质量文章
4. **数据驱动** — 每月复盘各平台数据，砍掉效果差的，加码效果好的
5. **不贪多** — 兼职做就专注 2 个平台，做深比做广重要
6. **开源是信任货币** — 开源项目的 star 和用户就是你的信用背书
