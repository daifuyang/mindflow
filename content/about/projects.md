# 项目档案

## 核心项目

### ZeroCMF / yishan

> 开源 CMS，帮助中小企业快速建站

- **仓库**: [github.com/daifuyang/yishan](https://github.com/daifuyang/yishan)
- **官网**: [zerocmf.com](https://www.zerocmf.com)
- **技术栈**: Next.js, TypeScript, Prisma
- **状态**: 活跃开发中
- **定位**: 通用内容管理解决方案，支持快速搭建企业级应用
- **历史版本**:
  - zerocmf-go（Go/Gin 版）
  - zerocmf-koa（Node.js/Koa 版）
  - 当前 yishan 为最新一代（Next.js 全栈）

### MindFlow

> Markdown 知识库，让 AI 更了解我

- **技术栈**: Next.js 16, React 19, Tailwind CSS v4, shadcn/ui
- **定位**: 个人知识库 + AI 上下文系统
- **目标**: 通过结构化的 Markdown 文档，让 AI 理解我的技术偏好、项目背景和业务目标，从而做出更精准的决策辅助

### Formily Admin

> 基于 Formily 的通用后台管理系统

- **仓库**: [github.com/daifuyang/formily-admin](https://github.com/daifuyang/formily-admin)
- **技术栈**: React, Formily, TypeScript
- **定位**: 利用现代 LLM 能力快速搭建应用的后台管理系统
